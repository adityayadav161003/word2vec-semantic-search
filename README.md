# 🧠 Word2Vec Semantic Search (Fake News Dataset)

An NLP project that uses **Word2Vec** to learn semantic relationships between words from a fake news dataset and perform **semantic similarity search**.

---

## 🚀 Features

- Text preprocessing using NLTK
- Word2Vec model training using Gensim
- Semantic word similarity search
- Word analogy using vector arithmetic
- Interactive CLI interface
